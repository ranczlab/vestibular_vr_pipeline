# BONSAI-rx
This is the software environment used for hardware and experimental control as well as recording. 

# HARP
This is the hardware platform used for control and most recordings. 

# ONIX
This is the hardware platform used for neuropixels recordings. 

# SETUP
## H1 board
HARP board located on the rotating platform with multiple input and output registers. 

## H2 board
HARP board located off the rotating platform with multiple input and output registers, chiefly responsible for motor control.
